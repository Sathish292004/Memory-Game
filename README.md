## 📖 Overview

Vault-Tec Memory Game is a browser-based card matching game where players test their memory skills by finding matching pairs of cards before time runs out.

The game includes multiple difficulty levels, score tracking, leaderboards, hints, move limits, and a themed Vault-Tec interface inspired by the Fallout universe.

## 🎮 Game Rules

1. Flip two cards at a time.
2. Match identical card pairs.
3. Complete all pairs before the timer expires.
4. Use fewer moves to earn more stars.
5. Challenge yourself with higher difficulty levels.
